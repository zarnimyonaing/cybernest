Malware/Virus/Trojan Awareness

Chapter Objective: Understand different types of malicious software and learn how to protect your devices from malware attacks.

What is Malware?

Malware (malicious software) is any software designed to harm, exploit, or gain unauthorized access to computer systems. Understanding different types of malware helps you recognize and prevent infections.

Types of Malware

Viruses

What it does: Attaches to legitimate files and spreads when those files are executed

How it spreads: Email attachments, infected software, USB drives

Damage: Corrupts files, slows system, deletes data

Trojans

What it does: Disguises as legitimate software but contains malicious code

How it spreads: Fake software downloads, email attachments

Damage: Steals data, creates backdoors, remote access

Worms

What it does: Self-replicating malware that spreads across networks

How it spreads: Network vulnerabilities, email, instant messaging

Damage: Network congestion, system crashes, data theft

Ransomware

What it does: Encrypts files and demands payment for decryption

How it spreads: Phishing emails, malicious websites, exploits

Damage: Data encryption, financial loss, business disruption

Common Signs of Malware Infection

Watch for these symptoms:
  • Slow computer performance
  • Frequent crashes or freezing
  • Unexpected pop-ups or advertisements
  • Changed browser homepage or search engine
  • Unfamiliar programs or files
  • High network activity when not in use
  • Disabled antivirus software
  • Missing or corrupted files

How Malware Spreads

Email Attachments

Malicious files sent via email

Software Downloads

Infected or fake software

Removable Media

USB drives, CDs, external drives

Protection Strategies

1. Antivirus Software

  • Install reputable antivirus software
  • Keep virus definitions updated
  • Run regular full system scans
  • Enable real-time protection

2. Operating System Updates

  • Install security patches promptly
  • Enable automatic updates
  • Keep all software updated
  • Remove unused programs

3. Safe Computing Habits

Safe Practices:
  • Verify email senders before opening attachments
  • Download software from official sources
  • Scan removable media before use
  • Use standard user accounts, not admin
  • Regular data backups
Avoid:
  • Clicking suspicious links or pop-ups
  • Downloading cracked or pirated software
  • Opening unexpected email attachments
  • Disabling security software
  • Using public computers for sensitive tasks

What to Do If Infected

Immediate Response Steps:
  1. Disconnect from the internet to prevent data theft and spread
  2. Boot from antivirus rescue disk or safe mode
  3. Run full system scan with updated antivirus software
  4. Remove detected threats following antivirus recommendations
  5. Change all passwords after cleaning the system
  6. Restore from clean backups if necessary
  7. Monitor accounts for unauthorized activity

Prevention is Key

The best defense against malware is prevention. By maintaining good security habits, keeping software updated, and staying vigilant, you can significantly reduce your risk of infection.

Remember: No security solution is 100% effective. A layered approach combining multiple protection methods provides the best defense against malware.
